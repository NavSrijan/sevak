## Intrduction
This is a project for my own use case. This is supposed to be the assistant that I have been trying to build for so long, what got me into programming. 

## Structure
`main.py` is the entry point of the project. 
`llm` directory contains the code for communication with the language model.
`skills` directory contains the code for the skills that the assistant can perform.
`finetuning` directory contains the code for finetuning the language model \[This is not exactly part of the project and only the output of the finetuning will be used here].
`comm` directory contains the code for communication with the user, attempting to integrate telegram here.
